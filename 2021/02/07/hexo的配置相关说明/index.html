hexo的配置相关说明

发表于 2021-02-07 更新于 2021-02-08
本文字数： 2.1k 阅读时长 ≈ 2 分钟

在上一篇中我们的博客已经部署完成了，那么该如何进行配置呢

首先弄清两个概念

站点配置文件：指的是你的博客根目录下的_config.yml文件

主题配置文件：指的是博客根目录下themes文件夹内，你下载的主题文件夹下的_config.yml文件

一、图片问题

1.管理好图片

可以使用图片托管，我喜欢保存在本地，使用如下方法

打开站点配置文件，找到post_asset_folder将其改为true

# Writing
new_post_name: :title.md # File name of new posts
default_layout: post
titlecase: false # Transform title into titlecase
external_link:
  enable: true # Open external links in new tab
  field: site # Apply to the whole site
  exclude: ''
filename_case: 0
nder_drafts: false
post_asset_folder: true  #改这里

然后Git Bash进入该目录，执行

npm install hexo-asset-image --save

如果不行则执行

npm install -g cnpm --registry=https://registry.npm.taobao.org

然后执行

cnpm install hexo-asset-image --save

此时执行hexo new “xxx”会同时在_post文件夹下生成同名文件夹，文件夹里放图片

如果上传后网页查看图片仍有问题，可以重新安装。使用此命令

npm install https://github.com/CodeFalling/hexo-asset-image --save

不行就用

cnpm install https://github.com/CodeFalling/hexo-asset-image --save

二、修改主题

1.安装主题

可以在 https://hexo.io/themes/ 查找自己喜欢的主题

个人觉得新手用 Next 比较好，官方教程比较详细，这里放出官方教程

https://theme-next.js.org/

https://theme-next.iissnan.com/getting-started.html

在你的blog文件下的themes文件夹里，使用git bash 

git clone https://github.com/iissnan/hexo-theme-next themes/next

2.启用主题

打开站点配置文件，找到theme字段，并将其值更改为 next

3.背景动画设置

我喜欢这个动画

（1）打开Git进入自己文件夹/themes/next文件夹下

（2）运行命令

git clone https://github.com/theme-next/theme-next-canvas-nest source/lib/canvas-nest

（2）详情可见 Hexo Next背景动画Canvas_nest 官方文件进行配置

首先在你的博客根目录下找到source文件夹

然后进入_date文件夹内（没有该文件夹就新建一个）

新建一个footer.swig文件，并编辑此文件添加以下内容

<script color="0,0,255" opacity="0.5" zIndex="-1" count="99" src="https://cdn.jsdelivr.net/npm/canvas-nest.js@1/dist/canvas-nest.js"></script>

自行修改参数，可以对动画自定义

（3）进入主题配置文件，取消footer行的注释

# Define custom file paths.
# Create your custom files in site directory `source/_data` and uncomment needed files below.
custom_file_path:
  #head: source/_data/head.swig
  #header: source/_data/header.swig
  #sidebar: source/_data/sidebar.swig
  #postMeta: source/_data/post-meta.swig
  #postBodyEnd: source/_data/post-body-end.swig
  footer: source/_data/footer.swig     #我在这里
  #bodyEnd: source/_data/body-end.swig
  #variable: source/_data/variables.styl
  #mixin: source/_data/mixins.styl
  #style: source/_data/styles.styl

成功部署网页后就有动画啦
