如何搭建个人博客

发表于 2021-02-07 更新于 2021-02-08
本文字数： 2k 阅读时长 ≈ 2 分钟
正经人谁写博客啊

   不喜欢广告，不喜欢某些网站设置的登录才能查看内容，于是决定自己搭建博客，本文使用hexo+github的方法使用个人电脑进行搭建

一、搭建环境的安装

1.Node.js下载和安装

https://nodejs.org/en/

2.Git下载和安装

https://git-scm.com/download/win

3.GitHub注册和创建仓库

https://github.com/

在GitHub账号创建成功之后，需要创建一个仓库来管理自己的文件。

(1) 点击个人主页右上角“+”里面的New repository

(2) 输入Repository name：用户名.github.io，注意：用户名一定要和注册的用户名一样

4.配置SSH免密登录

（1）在桌面点击鼠标右键，选择“Git Bash Here”，打开Git Bash终端。

（2）设置user name和email：

git config --global user.name 'GitHub用户名'
git config --global user.email '注册时候的邮箱'

查看是否配置成功

git config --list

（3）生成SSH密钥

ssh-keygen -t rsa -C 'GitHub注册邮箱'

输入命令后直接回车即可，在C盘的用户目录下会生成.ssh文件夹（我电脑上的目录为C:\Users\Misaka.ssh），里面有私钥id_rsa和公钥id_rsa_pub

（4）在GitHub上添加SSH keys

点击GitHub头像下的Settings->SSH and GPG keys->New SSH key，将本地的公钥id_rsa_pub内容复制粘贴到其中即可

（6）验证SSH免密是否成功

ssh -T git@github.com

二、安装Hexo

1.打开Git Bash，执行命令

npm install -g cnpm --registry=https://registry.npm.taobao.org   #备选源  http://registry.cnpmjs.org
cnpm install hexo-cli -g
cnpm install hexo --save

2.找一个合适的盘新建文件夹，起啥名随意，最好英文名如blog

3.在blog目录下点击鼠标右键，打开“Git Bash”终端，如果在其他地方打开Git Bash终端，会出现：hexo：commad not found 。初始化hexo：

hexo init

4.安装依赖及相关插件

cnpm install

5.启动Hexo

hexo server #或者hexo s

6.在浏览器中输入：localhost:4000，可以看到Hexo的示例页面

安装hexo时遇到的大部分问题是关于网络的，如果按照我的这一篇文章进行配置，一般都能成功安装

三、将博客部署到GitHub

1.配置文件以关联远程仓库，在blog目录下找到_config.yml文件，用文本编辑器Nodepad ++或Sublime Text打开，在最下方添加远程仓库地址：

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repository: git@github.com:yushengblog/yushengblog.github.io.git
  branch: master
#yushengblog改为你的github用户名（请修改）
#注意，如果repository写成https://github.com，在提交任务至Github时，会频繁输入用户名密码

2.新建博客，在E:\blog\source_posts目录下会生成firstblog.md的文件

hexo new "firstblog"

3.你可以根据Markdown语法来编辑上面的文件

推荐使用Visual Studio Code工具编辑，想要一边编辑一遍预览，可以使用Ctrl + Shift + P 调出主命令框，输入 Markdown，应该会匹配到几项 Markdown相关命令

4.部署博客的重要命令

（1）若部署后没有变化，可先执行以下命令再重新部署

 hexo clean

（2）生成静态页面至public目录（静态博客就在这个位置）

hexo generate #或 hexo g

（3）开启预览访问端口（默认端口4000，’ctrl + c’关闭server） http://localhost:4000/

hexo server #或 hexo s

（4）部署到GitHub

hexo d

现在登录你的博客查看吧
