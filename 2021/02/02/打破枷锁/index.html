小米路由器4A千兆版刷机

   去年双十一买了个小米4A千兆版的路由器，本来对路由方面一窍不通，却就此打开了新世界的大门。为了能实现ipv6的网络串流，软磨硬泡的找宽带师傅要到了移动光猫的超密（网上的密码一般都是初始超密，但超密会变的，所以最方便的还是找师傅要），之前询问客服和师傅，要求把光猫改为桥接，要不就说改不了，要不就说不会，好吧，自己动手，丰衣足食。

   其实改桥接挺简单的，这里就不多说了，网上都有教程，改前一定要在光猫配置页截图，一定要在光猫配置页截图，一定要截图！改完桥接以后就可以使用路由器拨号连接了，网速确实快了不少。然后我们就可以着手准备刷路由器了，其实我本来不想折腾的，可是小米这款路由器太坑了，没法正常的关掉防火墙，结果我就没办法通过公网ipv6串流到家里的电脑，当初不懂折腾了半天，坑！

   先感谢一下恩山无线论坛的大佬，下面正式开始刷机

   我是用的win10+ubuntu刷的，一定要确保全程用网线连接路由器

一、 安装ubuntu，可在win10商店中下载，我的是20.04.1版本

          为了方便后续操作，先进入管理员模式，输入sudo -i 然后输入密码即可

二、更换国内源

1. 在ubuntu中备份源文件（若已进入管理员模式，可删除sudo）

sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak

2. 编辑源文件

sudo vi /etc/apt/sources.list

   记得在英文模式下输入：按两下d键会删除一行，就这样删除所有内容，然后按i进入编辑模式，将下面清华大学的源复制进去

deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic-updates main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic-backports main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic-security main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic-security main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic-proposed main restricted universe multiverse
deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ bionic-proposed main restricted universe multiverse

   然后点ESC，输入 :wq! 然后回车即可

3. 更新源

sudo apt update

三、安装python3

sudo apt-get install python3.6
sudo apt-get install python3-pip

四、电脑连接小米路由器

   电脑浏览器打开小米路由器管理页面    192.168.31.1    不要关闭

五、在ubuntu中将代码拉到本地

   把 https://github.com/acecilia/OpenWRTInvasion 的代码利用下面的命令下载下来

git clone https://github.com/shenmeiqian/OpenWRTInvasion.git

六、利用漏洞开启telnet

cd OpenWRTInvasion
pip3 install -r requirements.txt
python3 remote_command_execution_vulnerability.py

   输入完之后系统会提示你输入路由器的ip 默认是192.168.31.1
回车之后会让你输入stok。这个stok就在上面我说的不让你关闭的小米路由器的管理页面。

   你地址栏显示的应该是 http://192.168.31.1/cgi-bin/luci/;stok=xxxxxxxxxx/web/home#router

   在地址栏复制stok，粘贴在ubuntu里面 然后回车。

   自动利用漏洞开启telnet，当显示telnet 192.168.31.1的时候一般来说就是成功了

七、telnet连接路由器，先备份eeprom(非常重要）

   这时候在linux里面输入

telnet 192.168.31.1

   提示你输入用户名此时输入 root 密码为 空 直接回车

   然后备份eeprom

dd if=/dev/mtd3 of=/tmp/eeprom.bin

八、获取breed

   小米4A千兆版 :

cd /tmp && wget  --no-check-certificate https://breed.hackpascal.net/breed-mt7621-pbr-m1.bin && mv breed-mt7621-pbr-m1.bin breed.bin

九、将备份的eeprom保存到本地

   下载winscp，可以在这里下载

https://pc.qq.com/detail/11/detail_2331.html

   安装好之后，参照下图连接路由器

   登陆后，右边找到tmp文件夹，点进去下载eeprom.bin到本地。（非常重要）

十、刷breed

mtd write breed.bin Bootloader

这一步请耐心等待，10分钟左右输入192.168.1.1即可访问

   要是无法打开，看看电脑的ip地址是不是自动获取，一般改为自动获取即可访问，否则可能就砖了

十一、刷固件

1. 先访问固件备份，两个选项都备份下，EEPROM、编程器固件 都需要备份一下，以免变砖；以防万一前往 MAC地址修改 截图所有MAC地址，前往固件更新添加备份的EEPEOM

2. 获取padavan固件

https://opt.cn2qq.com/padavan/MT7621_7603_7612_128M_3.4.3.9-099.trx

  在固件更新处，添加备份的EEPEOM和Padavan固件，确认无误后点击更新即可

  正常情况下进度条到头，即更新结束，保险起见可再等10分钟，然后重启路由器进入padavan后台

  管理页面：http://my.router/ 或 http://192.168.123.1/

  管理账号：admin

  密码：admin

  wifi密码:1234567890

  至此我们的路由器就调教好啦
